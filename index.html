Fall '21

A semester-long full-stack development project whose purpose is to emulate a recipe-sharing website for a number of users. Cryptune    
Fall '21

A little passion project which encodes messages as a series of tonal sounds, similarly to Morse code. The program is written in Python3, and generates tones of varying types by translating a series of points into a WAVE format. Once the tones of a certain message have been encoded and made playable, the program plays them aloud.

A game of Snake written in Python which plays on its own with relative proficiency using A* pathfinding and other AI algorithms. Uses PyGame to broadcast its progress to a display window as it operates.

Human Image Detector    
Spring '21

A MATLAB project in which an agent is trained using images containing a person versus images sans people by extracting HOG features from the images to recognize the general shape of a human. The agent, once trained, can identify people in a new image and draw a bounding box around each one.
